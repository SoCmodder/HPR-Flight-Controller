# HPR-Flight-Controller

## Branches
```  
  MASTER - Main branch. Currently being used for Arduino nano
  
  PICO-FULL - Main Branch for Raspberry Pi Pico
  
  PICO-FULL-WAVESHARE - Main Branch for Waveshare varient of Raspberry Pi Pico

  Waveshare Pins:
  PIN32 - 5v
  PIN31 - GND
  GPIO4 - SPI MISO
  GPIO5 - SPI CS
  GPIO6 - SPI SCK
  GPIO7 - SPI MOSI
  GPIO8 - I2C SDA
  GPIO9 - I2C SCL

  PIN39 - VSYS
  PIN38 - GND
```  
